# Estrutura do Código - Atividade HTML 

Este documento detalha a organização e as tags HTML utilizadas no arquivo `index.html` para a criação da página sobre o meu prato favorito (Empadão). O código foi estruturado de forma simples e semântica, atendendo aos requisitos solicitados na atividade.

## 📄 Tags HTML Utilizadas e Suas Funções

O código foi dividido e organizado utilizando as seguintes tags fundamentais:

* **`<!DOCTYPE html>` e `<html>`**: Definem o documento como HTML5 e configuram o idioma principal da página (`pt-BR`).
* **`<head>`**: Contém as configurações internas da página, como a codificação de caracteres (`UTF-8`) para aceitar acentos e o título da aba do navegador (`<title>`).
* **`<body>`**: Bloco principal que abriga todo o conteúdo visível que o utilizador final interage na página.
* **`<h1>`, `<h2>` e `<h3>`**: Elementos de títulos criados para estabelecer uma hierarquia visual clara entre o nome do aluno, as secções principais e os subtítulos.
* **`<hr>` (Horizontal Rule)**: Tag utilizada para criar as linhas horizontais que separam visualmente o conteúdo anterior, a descrição do prato e a receita.
* **`<p>`**: Utilizada para os blocos de texto e parágrafos descritivos.
* **`<strong>`**: Elemento de formatação usado para dar destaque e negrito a palavras-chave importantes, como o nome do prato e etapas principais.
* **`<ul>` e `<li>`**: Lista não ordenada (com marcadores de ponto) utilizada para listar os **Ingredientes** do prato.
* **`<ol>` e `<li>`**: Lista ordenada (numerada sequencialmente) utilizada para detalhar o passo a passo do **Modo de Preparo**, garantindo que a ordem das etapas seja seguida corretamente.

## 🛠️ Como Visualizar o Código no Navegador

1. Certifique-se de que o arquivo está salvo com o nome `index.html`.
2. Dê um duplo clique sobre o arquivo ou clique com o botão direito e selecione **"Abrir com"** seguido do seu navegador de preferência (Google Chrome, Microsoft Edge, Mozilla Firefox, etc.).
3. A página renderizará toda a estrutura do código localmente.
